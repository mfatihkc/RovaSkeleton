Progress Bar

rova-progress-bar is a custom built Rova component allows you to have a service controlled progress bar. It internally uses mat-progress-bar and provides a global service to control it.

Due to the nature of global progress bars, rova-progress-bar can only be used once in the entire app and it's currently located at app.component.html file.
